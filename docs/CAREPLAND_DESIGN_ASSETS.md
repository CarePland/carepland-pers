# CarePland Design Assets

Purpose: shared user-facing language patterns that make CarePland feel calm, conversational, and less system-like.

Use the better pair when possible, especially in patient-facing UI.

| Concept | Better Pair |
| --- | --- |
| Accurate / Inaccurate | Looks Right / Not Quite |
| Related / Unrelated | Related / Separate |
| Save Feedback | Add Your Detail |
| Submit Correction | Share Anything Different |
| Open Story / Close Story | Story Open / Close |
| Health Themes | Health Focus |
| Context Analysis | Past Visit Context |

These are vocabulary preferences, not rigid legal strings. If a surface needs a variation, keep the same tone: plain, human, and action-oriented.
